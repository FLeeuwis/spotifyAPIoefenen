<!-- INSTALLATION -->

### Installatie

#### React

1. clone de repo

```sh

    git clone https://github.com/FLeeuwis/spotifyAPIoefenen

```

2. Instaleer de NPM packages

   ```sh

   npm install

   ```

3. Zorg ervoor dat de react app werkt doormiddel van run dev

   ```sh

   npm start

   ```
